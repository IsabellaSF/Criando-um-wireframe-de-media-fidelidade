# Criando um wireframe de média fidelidade

Este projeto apresenta a estrutura de uma landing page desenvolvida em média fidelidade, focando na hierarquia visual, disposição de elementos e fluxo de navegação antes da aplicação da identidade visual final.

## 1. Estrutura da Página
Com base no wireframe desenvolvido, a página está dividida nas seguintes seções:

* **Cabeçalho (Header):** Menu de navegação centralizado com os links "Work", "About" e "Contact".
* **Seção Hero:** Título principal (H1) à esquerda, seguido de texto de apoio e um botão de Chamada para Ação (Call to Action - CTA). À direita, um espaço reservado (placeholder) para imagem ou ilustração principal.
* **Seção de Conteúdo Secundária:** Layout invertido para criar ritmo visual ("scroll-stopper"), com imagem à esquerda e texto com botão à direita.
* **Rodapé (Footer):** Dividido em colunas contendo informações institucionais, ícones de redes sociais (Instagram, LinkedIn, X) e links de navegação para funcionalidades, aprendizado e suporte.

## 2. Características Técnicas do Modelo
* **Paleta de Cores:** Utilização de tons de cinza para neutralidade estética, permitindo foco total na funcionalidade.
* **Tipografia:** Uso de fontes sem serifa para garantir legibilidade técnica.
* **Componentes:** Botões com cantos arredondados e preenchimento sólido em azul para destacar as ações principais (CTAs).
* **Placeholders:** Espaços cinzas sólidos representam onde serão inseridos os ativos visuais (imagens/vídeos) na fase de alta fidelidade.
